# 날씨정보,대기오염 정보 제공 및 알람

## 기술스펙 : Node.js & Express.js / Angular.js / CANVAS / HTML5 / CSS3

## API : SK - Weather Planet / 한국환경공단 - 대기오염정보 조회 서비스

## 네이티브 기능 연동 : GPS, 시계

### 1차 스펙

1. Intro - GPS 허용 요청 알림 뜸. 거부시 앱 종료
2. 메인 - 현재 기온, 기상, 미세먼지표시. 금일 시간대별 날씨 표시, 금주 요일별 날씨 표시. 우측 상단에 검색 아이콘 위치.
3. 검색 - 지도에서 지역 검색하여 날씨
4. 메뉴 - 위치추가
5. 메뉴 - 알람
6. 메뉴 - 이용약관
7. 메뉴 - 앱 정보 - 앱,개발자 정보 및 메일 보내기
